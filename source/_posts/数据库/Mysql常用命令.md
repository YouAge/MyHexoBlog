MySql 常用命令的基本使用篇章


## 初始操作


```sql
-- 查看所有数据库 
show databases;
-- 选择数据库
use db;
-- 查看到但钱使用的数据库
select database();
-- 创建数据库
create database dbname charset=utf8;
-- 查看当前数据库中所有的表
show tables;
-- 查看表结构
desc 表名;

-- sql格式数据导入数据
insert into 表名 values (数据)

```
